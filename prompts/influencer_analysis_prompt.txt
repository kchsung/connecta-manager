인플루언서 분석을 위한 AI 프롬프트

다음 인플루언서 데이터를 분석하여 종합적인 평가를 제공해주세요:

**분석 대상 데이터:**
- 인플루언서 기본 정보 (이름, 팔로워 수, 팔로잉 수, 게시물 수)
- 팔로워 네트워크 분석 데이터
- 댓글 진정성 분석 데이터
- 콘텐츠 분석 데이터

**분석 요구사항:**

1. **팔로워 네트워크 분석 (follow_network_analysis)**
   - 팔로워의 품질과 진정성 평가
   - 팔로워 증가 패턴 분석
   - 팔로워와의 상호작용 품질

2. **댓글 진정성 분석 (comment_authenticity_analysis)**
   - 댓글의 진정성과 품질 평가
   - 스팸 댓글 비율 분석
   - 의미있는 상호작용 비율

3. **콘텐츠 분석 (content_analysis)**
   - 콘텐츠 품질과 일관성 평가
   - 콘텐츠 주제의 다양성
   - 브랜드 적합성 분석
   - inference_confidence: 분석 신뢰도 (0-1 사이 값)

4. **종합 평가 (evaluation)**
   - engagement: 참여도 점수 (0-10)
   - activity: 활동성 점수 (0-10)
   - communication: 소통 능력 점수 (0-10)
   - growth_potential: 성장 잠재력 점수 (0-10)
   - overall_score: 종합 점수 (0-10)

5. **인사이트 (insights)**
   - 주요 강점과 약점
   - 브랜드 협업 적합성
   - 타겟 오디언스 분석

6. **요약 (summary)**
   - 핵심 분석 결과 요약 (200자 이내)

7. **추천 (recommendation)**
   - 다음 중 하나로 선택: "매우 추천", "추천", "보통", "비추천", "매우 비추천"

8. **추가 노트 (notes)**
   - 특별한 주의사항이나 추가 정보

**출력 형식:**
JSON 형태로 응답해주세요. 모든 필드는 반드시 포함되어야 합니다.

**분석 기준:**
- 데이터의 신뢰성과 일관성
- 브랜드 마케팅 관점에서의 실용성
- 인플루언서의 장기적 성장 가능성
- 타겟 오디언스와의 매칭도
